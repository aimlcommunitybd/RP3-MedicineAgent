# Git Basics

There are most used git commands in a project.

- `git clone <repository>`: clone repo to local
- `git branch`: shows current branch with star and list of other branches
- `git checkout -b <branch-name>`: create a new branch and move to that branch
- `git checkout .`: delete all unstaged changes
- `git status`: check changes
- `git add <changed file>`: stage modified/new/deleted file
- `git add .`: stage all changes
- `git commit -m"<commit message>"`: commit staged changes
- `git commit -a -m"<commit message>"`: stage all changes and commit 
- `git log`: list of commits
- `git push`: push changes to repote repository (github)
